Project Overview

The Wine Quality Prediction project aims to predict the quality of wine based on its physicochemical properties. The project leverages machine learning algorithms to analyze and predict wine quality, which is crucial for winemakers and consumers alike. The dataset used in this project contains various features such as acidity, pH, alcohol content, and more, which are used to determine the quality of the wine.

Objectives

  • To predict the quality of wine using machine learning models.
  • To compare the performance of different machine learning algorithms.
  • To provide an interactive web interface for users to input wine features and get quality predictions.
  • To visualize the model's performance using various metrics such as accuracy, confusion matrix, and ROC curves.

Methodology

The project follows a structured approach to build and evaluate the wine quality prediction model:

  1. Data Preprocessing: Handling missing values, normalizing labels, and addressing class imbalance using SMOTE.
  2. Feature Engineering: Dropping highly correlated features and standardizing the dataset.
  3. Model Selection: Testing multiple machine learning models including Logistic Regression, Random Forest, XGBoost, LightGBM, and SVM.
  4. Hyperparameter Tuning: Using Optuna for hyperparameter optimization to improve model performance.
  5. Model Evaluation: Evaluating the models using cross-validation, accuracy, confusion matrix, and classification reports.
  6. Deployment: Deploying the best-performing model using Flask to create a web application.
